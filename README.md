# Image Cropping Tool

一个简单易用的图片裁剪工具，可将长图裁剪成指定数量的短图片。

## 功能特点

- 支持 JPG、PNG 格式图片
- 可视化操作面板
- 自定义裁剪尺寸
- 保持原始比例，不添加填充
- 图片压缩功能
- 批量下载裁剪结果
- 支持使用原图宽度选项，裁剪数量根据高度自动计算

## 如何运行

1. 安装依赖：
   ```
   npm install
   ```

2. 启动开发服务器：
   ```
   npm run dev
   ```

3. 在浏览器中打开 http://localhost:5173 查看应用
